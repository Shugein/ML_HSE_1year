# Прогнозирование стоимости автомобилей с помощью регрессионных моделей

## Описание проекта

Данный проект представляет собой комплексное решение задачи машинного обучения по предсказанию рыночной стоимости подержанных автомобилей. В рамках работы реализован полный цикл разработки ML-модели: от первичного анализа данных до создания интерактивного веб-приложения для прогнозирования цен.

## Цели проекта

- Построить точную модель машинного обучения для оценки стоимости автомобилей
- Провести глубокий анализ факторов, влияющих на ценообразование
- Разработать пользовательский интерфейс для работы с моделью
- Создать воспроизводимый и хорошо документированный исследовательский процесс

## Структура работы

### Часть 1: Разведочный анализ данных (EDA)

**Базовый анализ данных (1.7 балла)**
- Изучение структуры датасета и типов данных
- Обнаружение и обработка пропущенных значений
- Поиск дубликатов в данных
- Генерация автоматического отчета с помощью ydata-profiling
- Формулирование первичных выводов о данных

**Визуализация и углубленный анализ (1.6 балла)**
- Построение информативных графиков распределений
- Анализ взаимосвязей между признаками
- Выявление аномалий и выбросов
- Исследование категориальных переменных

### Часть 2: Подготовка данных

**Препроцессинг признаков**
- Обработка текстовых полей (mileage, engine, max_power, torque)
- Нормализация числовых характеристик
- Кодирование категориальных переменных
- Создание новых информативных признаков (Feature Engineering)
- Применение робастного масштабирования для устойчивости к выбросам

### Часть 3: Моделирование

**Обучение базовых моделей**
- Линейная регрессия как baseline
- Ridge регрессия с подбором гиперпараметров
- Валидация моделей с использованием кросс-валидации

**Улучшение качества**
- Логарифмическое преобразование целевой переменной
- Полиномиальные признаки
- Оптимизация параметров регуляризации
- Сравнение различных подходов к обработке данных

### Часть 4: Оценка моделей

**Стандартные метрики**
- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)
- R² Score

**Кастомные бизнес-метрики**
- Доля прогнозов с точностью в пределах 10% от реальной цены
- Метрика потенциальных убытков с учетом асимметричности ошибок
- Анализ недо- и перепрогнозов

### Часть 5: Интерактивное приложение (3 балла)

Разработано Streamlit-приложение с функционалом:
- **Визуализация данных**: интерактивные графики и гистограммы из EDA
- **Прогнозирование**: возможность загрузки CSV-файла или ручного ввода характеристик автомобиля
- **Интерпретация модели**: визуализация весов признаков и их влияния на предсказания

### Часть 6: Документация и развертывание (3 балла)

## Технологический стек

- **Python 3.12**: основной язык программирования
- **Pandas & NumPy**: обработка и анализ данных
- **Scikit-learn**: построение ML-моделей
- **Matplotlib & Seaborn**: визуализация данных
- **ydata-profiling**: автоматическая генерация EDA-отчетов
- **Streamlit**: создание веб-интерфейса
- **Pickle**: сериализация обученных моделей
